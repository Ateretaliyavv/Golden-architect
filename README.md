# Golden-architect - עטרת ון לואן וטליה ולרשטיין
"אדריכל הזהב"

**משחק בבניית בתים, האם תצליח לעמוד בתנאים הדרושים ובתוצאה זהה?**

## מהות המשחק

משחק שלבים בו נדרש השחקן לבנות בתים באופן יציב וחזק ובצורה נכונה בהתאם לחוקי הבנייה באיזור הנתון, בהנתן תמונה ותנאי שטח, ומשאבי בנייה דרושים. על השחק להצליח לבנות מבנה זהה ויציב. ע"י צבירת נקודות יתקדם השחקן לשלבים הבאים.
המשחק מתאים למחשב רגיל ולפלאפון.
